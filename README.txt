RangeHTTPRequestHandler is a request handler that is written over BaseHTTPRequestHandler in the package http.server.

It supports Range header.

This is a python 3 port of user smgoller's rangehttpserver
project link : https://github.com/smgoller/rangehttpserver/

usage :
$ python RangeHTTPRequestHandler.py

The server by default runs on localhost:9999 and can be changed by editing the string
